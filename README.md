## 📝 Overview

This project is a high-performance clone of MyAnimeList, developed using Angular and Tailwind CSS. It offers an enhanced user experience with improved performance and responsive design compared to the original platform, allowing users to browse, search, and track anime and manga with ease.

> **Disclaimer**: This project is not affiliated with, endorsed by, or connected to MyAnimeList.net. It was created for educational purposes only.

### 🎯 Features

- **Browse Anime/Manga**: Explore a vast collection of anime and manga titles
- **Advanced Search**: Find specific titles quickly with powerful filtering options
- **Rating System**: Review anime and manga with an intuitive interface
- **Superior Performance**: Optimized loading times and smoother interactions
- **Enhanced Mobile Experience**: Fully responsive design with mobile-first approach

### 🚀 Performance Improvements

This clone offers significant improvements over the original MyAnimeList platform:

- **Faster Load Times**: Optimized assets and lazy-loading for quicker page rendering
- **Smoother Animations**: Efficient state management for seamless transitions
- **Reduced API Calls**: Smart caching and request batching for better performance
- **Mobile Optimization**: Superior responsiveness on all device sizes
- **Accessibility**: Enhanced screen reader support and keyboard navigation

### 🛠️ Technologies Used

- **Angular v17.3.12**: Frontend framework with modern rendering engine
- **Tailwind CSS**: Utility-first CSS framework for responsive design
- **TypeScript**: Strongly-typed programming language
- **HTML/CSS**: Semantic markup and styling
- **RxJS**: Reactive programming library for asynchronous operations

### ⚙️ Installation and Setup

1. Clone the repository:

```bash
git clone https://github.com/oliveiravitor32/MyAnimeList_Clone-Angular.git
```

2. Navigate to the project directory:

```bash
cd MyAnimeList_Clone-Angular
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
ng serve
```

5. Open your browser and navigate to `http://localhost:4200`

### 📚 API Integration

This project uses [Jikan API (4.0.0)](https://jikan.moe/) to fetch anime and manga data with optimized request handling to prevent rate limiting and improve performance.
